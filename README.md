# Installation

First install dependencies, go to either the backend or client folder and run:
### `npm run bio`

This will install all dependencies, as well as run the site for the first time.
For any subsequent runs, please use:
### `npm run dev`

This will start up both the backend development server and the client rendering
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

The page will reload if you make edits.\
You will also see any lint errors in the console.

Happy Testing!
